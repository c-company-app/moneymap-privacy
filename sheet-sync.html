ตั้งค่า Sync ขึ้น Google Sheet

ใช้เวลา ~3 นาที · ทำครั้งเดียว · ไม่ต้อง login ในแอป

ข้อมูลจะถูกส่งไปที่ Google Sheet ของคุณเอง ผ่านสคริปต์ที่คุณ deploy ในบัญชี Google ตัวเอง — ไม่ผ่านเซิร์ฟเวอร์ของ MoneyMap และแอปไม่ขอ login เข้า Google เลย

  1. สร้าง Google Sheet ใหม่

    ไปที่ sheets.new เพื่อสร้างสเปรดชีตเปล่า ๆ (ตั้งชื่ออะไรก็ได้ เช่น “MoneyMap”)

  2. เปิด Apps Script

    ในชีตนั้น เลือกเมนู Extensions → Apps Script จะเปิดหน้าต่างแก้โค้ดขึ้นมา

  3. วางโค้ดนี้ทับทั้งหมด แล้ว Save

    ลบโค้ดเดิม (function myFunction…) แล้ววางโค้ดด้านล่างแทน จากนั้นกดรูปแผ่นดิสก์ (Save)

    
        
    
      
    4. 
      
  4. 
        
    Deploy เป็น Web app
    
        
    กด Deploy → New deployment → ไอคอนเฟือง เลือก Web app
    
        
    ตั้งค่า: Execute as = Me · Who has access = Anyone → กด Deploy
    
        
    ครั้งแรก Google จะขอสิทธิ์ → กด Authorize / อนุญาต (เป็นบัญชีคุณเอง)
    
      
    5. 
      
  5. 
        
    คัดลอก URL แล้ววางในแอป
    
        
    คัดลอก Web app URL (ลงท้ายด้วย /exec)
    
        
    ในแอป MoneyMap → Export → หัวข้อ Sync ขึ้น Google Sheet → URL ปลายทาง → วาง URL → บันทึก → กด Sync
    
      
    6. 
    

    
คำถามที่พบบ่อย

    

      
ข้อมูลปลอดภัยไหม? ชีตอยู่ในบัญชี Google ของคุณ คุณควบคุม/ลบเองได้ · URL เป็นความลับ อย่าเผยแพร่ (ใครมี URL ก็ส่งข้อมูลเข้าชีตได้)

      
Sync แล้วข้อมูลซ้ำไหม? ไม่ซ้ำ — แอปส่ง “ทุกรายการ” ทุกครั้ง แล้วสคริปต์เขียนทับแท็บ “MoneyMap” ให้ตรงกับแอปเสมอ

      
อยากทำ Pivot/สรุป? อย่าแก้แท็บ “MoneyMap” โดยตรง — สร้างแท็บใหม่แล้วอ้างอิง 'MoneyMap'!A:K

      
แก้โค้ดแล้ว sync ไม่เปลี่ยน? ต้อง Deploy → Manage deployments → Edit → New version ทุกครั้งที่แก้โค้ด

    

  


  
  

    
Set up Sync to Google Sheet

    
~3 minutes · one-time · no in-app login

    
Your data goes to your own Google Sheet, via a script you deploy under your own Google account — never through MoneyMap’s servers, and the app never asks you to log into Google.

    
    
      
  1. 
        
    Create a new Google Sheet
    
        
    Go to sheets.new to make a blank spreadsheet (name it anything, e.g. “MoneyMap”).
    
      
    2. 
      
  2. 
        
    Open Apps Script
    
        
    In that sheet, choose Extensions → Apps Script. A code editor opens.
    
      
    3. 
      
  3. 
        
    Paste this code over everything, then Save
    
        
    Delete the default code (function myFunction…), paste the code below, then click the disk icon (Save).
    
        
    
          
          
    
        
    
      
    4. 
      
  4. 
        
    Deploy as a Web app
    
        
    Click Deploy → New deployment → gear icon, choose Web app.
    
        
    Set Execute as = Me · Who has access = Anyone → click Deploy.
    
        
    The first time, Google asks for permission → Authorize (it’s your own account).
    
      
    5. 
      
  5. 
        
    Copy the URL and paste it into the app
    
        
    Copy the Web app URL (ends in /exec).
    
        
    In MoneyMap → Export → Sync to Google Sheet → Destination URL → paste → Save → tap Sync.

FAQ

    

      
Is it safe? The sheet lives in your Google account; you control and can delete it. Keep the URL secret — anyone with it can write to your sheet.

      
Will syncing duplicate rows? No — the app sends “all records” each time and the script overwrites the “MoneyMap” tab to always match the app.

      
Want pivots/summaries? Don’t edit the “MoneyMap” tab directly — make a new tab referencing 'MoneyMap'!A:K.

      
Edited the code but sync didn’t change? You must Deploy → Manage deployments → Edit → New version each time you change the code.
